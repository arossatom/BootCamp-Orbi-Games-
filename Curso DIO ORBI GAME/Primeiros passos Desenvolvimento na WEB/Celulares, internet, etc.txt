# Celular, internet e outros dispositivos

## Dados movéis

- ## SMS

- Quanto custa para uma operadora de telefonia móvel, enviar um SMS?

  Nada. O celular troca, naturalmente, alguns bits com as torres de comunicação.

- ## MMS

- Transmissão de mensagens multimídia (áudio, vídeo)  por meio de uma espécie de conexão de dados primitiva.

- ## Conexões Móveis

- 1G (analógico): 10Kbps

- 2G (digital) (GSM (Global System for Mobile Communication)): 97 Kbps

- GPRS (General Packet Radio Service) / 2,5G: 32-80Kbps - dados+voz

- EDGE (Enranced Data Rates for GSM Evolution) / 2,75G: 128-236 Kbps

- 3G (7Mbps), 4G (22,1Mbps), 5G (10Gbps)



- ## WIFI

- IEEE 802.11:2,4 GHz, 2Mbps

- IEEE 802.11a: 5GHz, 54Mbps

- IEEE 802.11b: 2,4GHz, 11Mbps (diminuição de interferencia)

- IEEE 802.11g: 2,4GHz, 54Mbps

- IEEE 802.11n: 2,4GHz/5GHz, 150-600Mbps

- WEP: chaves (senhas) de 64 bits e de 128 bits

- WPA: chave trocada periodicamente

- WPA2 (AES) (802.11i)

  - +segurança

  - +processamento
